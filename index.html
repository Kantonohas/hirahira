☁️ Base de données en ligne (GitHub Gist)

Connectez GitHub Gist pour partager vos chansons entre tous vos appareils. 100% gratuit, open source, sans quotas.

📋 Comment faire (gratuit, open source, sans quotas) :
  1. Allez sur github.com et créez un compte gratuit
  2. Allez sur gist.github.com, créez un nouveau Gist public ou secret avec le fichier nommé songs.json et le contenu {"songs":[]}
  3. Copiez l'ID du Gist depuis l'URL (ex: abc123def456...)
  4. Allez dans Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token avec la permission gist cochée
  5. Collez les deux valeurs ci-dessous et cliquez Enregistrer
